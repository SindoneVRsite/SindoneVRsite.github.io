# Summary

This a model of a fictional small corvette. She would be armed with 2 quad 30mm CIWS mounts and a number of multi-purpose VLS tubes.

I flattened out the bottom so it prints easy. 

If you print the main guns and radar separate from the hull (see files provided), no supports are needed. Chances are high that the railings won't print....they are just for show.
